# 🗳️ 2024 Indian Lok Sabha Election Data Analysis

## Project Overview

This project analyzes the results of the 2024 Indian Lok Sabha Election using Python. The analysis includes data cleaning, exploratory data analysis (EDA), and visualizations to identify trends in party performance, victory margins, and constituency-level outcomes.

## Objectives

- Analyze seat distribution among political parties.
- Examine victory margins across constituencies.
- Identify the closest and largest victory margins.
- Compare the performance of winning and runner-up parties.

## Dataset

- **Dataset:** 2024 Indian Lok Sabha Election Results
- **Records:** 543 constituencies

## Visualizations

- Distribution of Victory Margins

## Key Findings

- Major national and regional parties dominated the election.
- Victory margins varied significantly across constituencies.
- Some constituencies witnessed extremely close contests, while others recorded decisive victories.
